# Giới thiệu

- Đây là dự án học React, sản phẩm là 1 blog để nhiều người cùng kết nối và chia sẻ kiến thức.  

# Tính năng

1. Đăng ký
2. Đăng nhập
3. View profile
4. Chỉnh sửa profile
5. Viết blog mới
6. Chỉnh sửa blog cũ
7. View danh sách blog của bản thân
8. View blog của người khác
9. Xóa blog
10. Comment
11. Like
12. Share Facebook

# Công nghệ

- Front-end: 
    - React
    - Bootstrap
- Back-end:
    - Node.js
    - Express.js
    - MongoDB-moongoose
